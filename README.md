## ✨ Introduction
**Bookified** is an AI-powered platform that allows you to have real-time voice conversations with your PDF books. Built with Next.js 15, Vapi, and ElevenLabs, it utilizes Retrieval-Augmented Generation (RAG) to transform static documents into interactive entities. Whether you need a summary or want to dive deep into specific chapters, Bookified makes it possible through natural voice synthesis.

## ⚙️ Tech Stack
* **Framework**: Next.js 15 (App Router)
* **Authentication**: Clerk
* **Voice AI**: Vapi (Real-time conversational audio)
* **Speech Synthesis**: ElevenLabs (Lifelike AI voices)
* **AI Engine**: Google Gemini AI (Embeddings & Content Generation)
* **Storage**: Vercel Blob (PDF file hosting)
* **Database**: MongoDB (Metadata & user library management)
* **UI/UX**: Tailwind CSS & Shadcn UI
